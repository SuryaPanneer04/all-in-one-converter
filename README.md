# 🚀 All-in-One File Converter

A powerful and user-friendly web application to convert **video, audio, image, and document files** using PHP, FFmpeg, and LibreOffice.

---

## ✨ Features

### 🎥 Video
- Convert between formats (MP4, AVI, MOV, MKV, WEBM, etc.)
- Video compression (HD quality)
- Change resolution (1080p, 720p, 480p)
- Video to Audio (MP3, WAV)

### 🎵 Audio
- Convert between audio formats
- Extract audio from video

### 🖼️ Image
- Convert between image formats
- Image optimization

### 📄 Documents
- PDF → Text
- Word (DOC/DOCX) → PDF
- PDF → PDF compression

---

## 🛠️ Tech Stack

- HTML
